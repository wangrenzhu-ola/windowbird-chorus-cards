# App Store Metadata Draft

## Name
WindowBird Chorus Cards

## Subtitle
Private window birdsong cards

## Promotional Text
Turn a two-minute window listen into a private card with rhythm, direction, weather, and mood.

## Description
WindowBird Chorus Cards helps beginner bird listeners build a gentle sound map without pressure to identify species. Open Morning Chorus, choose the sound shape you heard, add direction, weather, mood, an optional window view photo, and a private note, then review saved listens as neighborhood sound dots.

Listen cards, window view photos, and notes stay on this device. The listening flow does not require microphone recording. Premium Dawn Pack is an optional content pack for extra visual themes and sticker roosts; limited tracking may be used only to understand whether Premium offers feel helpful.

## Keywords
birds, nature, backyard, journal, sound, morning, map, mindfulness

## Privacy Summary
Listen cards, window view photos, and notes stay on this device. The app declares photo library, camera, microphone, and tracking permission descriptions for window view photos, optional birdsong listening helpers, and Premium offer measurement; the core listening flow does not prompt for microphone access. The app has no login and no backend service for this version.
