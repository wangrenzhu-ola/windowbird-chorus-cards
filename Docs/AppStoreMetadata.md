# App Store Metadata Draft

## Name
WindowBird Chorus Cards

## Subtitle
Private window birdsong cards

## Promotional Text
Turn a two-minute window listen into a private card with rhythm, direction, weather, and mood.

## Description
WindowBird Chorus Cards helps beginner bird listeners build a gentle sound map without pressure to identify species. Open Morning Chorus, choose the sound shape you heard, add direction, weather, mood, and an optional private note, then review saved listens as neighborhood sound dots.

No microphone recording is required. Optional notes stay private on device. Premium Dawn Pack is an optional content pack for extra visual themes and sticker roosts; the core listening flow remains free.

## Keywords
birds, nature, backyard, journal, sound, morning, map, mindfulness

## Privacy Summary
No microphone recording is required; optional notes stay private on device. The app has no login and no backend service for this version.
